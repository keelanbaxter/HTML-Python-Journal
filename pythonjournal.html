The Syntax of Python

Comments & Variables

  • Comments: #This is a Python comment. You can also do multiple lines of one comment, by either starting each line with a #, or, by using 6 quotation marks. (""") 3 go above the comment, and then another 3 go below the comment.

  • Variables 1: These are undefined in Python, and can be anything. For example, X = Keelan or Y = Baxter. Now, when our code editor reads the Variables 'X' or 'Y', we will get Keelan, or Baxter. To print a variable to the screen, say print (x) or print (y) It's as easy as that. When x = "Keelan", this variable becomes a 'string', as it has two quotation marks around it.

  • Variables 2: On one line of code, you can write/define multiple variables. Example:) x, y, z = "Red", "Blue", "Green" can all be on the same line.

  • Variables 3: Furthermore, another feature of Python variables, is that you can display multiple variables by only having to type out one variable. Example:) z = x + y. By telling the computer to print (z), it will also print both x, and y variables together. This can save a lot of time, instead of having to type out each individual variable. In the same sense, you can say, print (z + "is cool"), to have x, y variables printed, plus a line of text to go along with it. Sort of like a World of Warcraft macro.

Data Types

Within the world of programming, data types become important. Data Types and Variables work together. Variables, store different types of data - and different data, does different things. So to figure out which variable contains which style of data, we have data types, as follows:

  • Text Type: str

  • Numeric Types: int, float, complex

  • Sequence Types: list, tuple, range

  • Mapping Type: dict

  • Set Types: set, frozenset

  • Boolean Type: bool

  • Binary Types: bytes, bytearray, memoryview
  • str: x = "hello user"

  • int: x = 20
  • float: x = 20.5
  • complex: x = 1j

  • list: x = ["red", "blue", "green"]
  • tuple: x = ("red", "blue", "green")
  • range: x = range(6)

  • dict: x = {"name" : Keelan, "age" : 23}

  • set: x = {"red", "blue", "green"}
  • frozenset: x = frozenset ({"red", "blue", "green"})

  • bool: x = true

  • bytes: x = b"Hello"
  • bytearray: x = bytearray(5)
  • memoryview: x = memoryview (bytes (5))
