The Numbers of Python

In the last page, I covered the data-types of Python. One of these data-types are called 'Numeric types'. There are 3 numeric types in Python.

  • int
  • float
  • complex
  • x = 1 (int)
  • y = 2.8 (float)
  • x = 1j (complex)

Given this, variables of their respected numeric types are then created once a value is assigned to them. So, for example - if I were to create a variable, keelan = 5, then that variable would be then classified as an 'int' numeric type. Whereas, if I were to make a variable, keelan = 3.5, then the variable would be classified as a'float' numeric type. And with the variable keelan = 5b, it would be classified as a 'complex' numeric type.

To verify a specified type of any object within the Python language, write the 'type()' function. This function provides the definition of what each variable is classified as, when print (type (keelan) ) is written.
Given the variable is 'keelan', whatever numeric value is assigned after, the function will tell me what type of numeric the variable is - being an int, float or complex.


Int

Int, stands for interger. This is a whole number, positive or negative. Int has no decimals involved, and has an unlimited length.

Examples of Int (intergers):

  • x = 1
  • y = 2565222554887771 (this works as long as they're whole numbers.)
  • z = -3522771 (as above, is below here. An 'int' numeric type can be a long list of negative numbers, so long as they're whole numbers.

  • Don't forget the function: print (type (x))

Float

Float is basically a fancy way of saying a number with a decimal. Float stands for "floating point number", and is classified as a positive or negative number containing one, or more decimals.

Within the Float type, we can place an 'e' beside the number. This will define it's power, say, if we wanted to square a number. E.g.) x = 8e2, or y = 12E4, even z = -87.7e100.

Examples of Floats:

  • x = 1.15
  • y = 1.30
  • z = -25.59

  • Using the function print (type (x)), the console/terminal will display what this variable is classified as a numeric type.

Complex

The numeric type known as 'Complex', simply means that, any number assigned to a variable with the letter J beside it. E.g.) x = 3+5j. The variable has a 'j' beside a number. The 'j', means an imaginary value. Anytime there is the letter 'j' beside a number within a variable, it will be recongized as a complex numeric type.

Type Conversion

Within the Python Language, we can convert from one numeric data type, to another. To do so, type: int(), float(), or complex(). Say, if a variable (x = 1), which would be an int, we could convert that into a float, by typing a = float(x).

More examples of conversion:

  • x = 1 # int
  • y = 2.8 # float
  • z = 1j # complex

  • #convert from int - float: a = float(x)
  • #convert from float - int: b = int(y)
  • #convert from int - complex: c = complex(x)

  • Complex cannot be converted into anything - however, int and float can be converted into complex.

Random Number

I can also generate random numbers in the Python Programming Language. There is no function, such as 'random()', like there is with other functions. So, to make a random number, we use the built in module called 'random'.

To use the module 'random', we import it. Meaning, we type:

  • 1.) import random
  • 2.) print (random.randrange(1,10) #The 'rand' stands for random. This can also be between 5-10, 1-100, or 9-1000, for example.
